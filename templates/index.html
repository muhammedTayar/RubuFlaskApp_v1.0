Produktbeschreibung

Das Produkt Temas ist ein Mini-Gimbal mit mehreren Funktionalitäten, welche anhand der eingebauten Komponenten ermöglicht werden können. Eine Kamera, ein Lidar-Laser und zwei Motoren können über eine Wifi-Schnittstelle angesteuert werden. Eine mitgelieferte Software erlaubt die Verwendung folgender Funktionen:

Bewegung der Motoren,

Live-Stream Kamera ,

Entfernungsmessung,

3D-Ansicht,

Point-Cloud,

Objektverfolgung,

Positionskontrolle von Objekten und

das Scannen von QR-Codes.

Des Weiteren wird eine API freigegeben um eigene Ideen im Bereich der künstlichen Intelligenz bzw. maschinelles Sehen und Deep Learning umzusetzen. Die Objektverfolgung ermöglicht eine 3D Trajektorie mit dem physikalische Experimente dem Digitalen Zeitalter entsprechend aufgezeichnet und ausgewertet können. In eine Prozessoptimierung im neuen Standard der Industrie 4.0 lässt sich das Produkt Temas leicht integrieren. All diese Eigenschaften und vieles mehr sind in dem Produkt Temas vereint!
